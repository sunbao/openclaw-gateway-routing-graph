# OpenClaw Gateway Routing Graph 2026.3.16

首个可公开使用的版本，聚焦 OpenClaw 网关链路的独立可视化、中文业务视图和可单独部署的监控能力。

## Highlights

- 独立项目形态，不修改 OpenClaw 主工程源码
- 中文优先的业务驾驶舱界面
- 实时链路图，默认显示骨架，有数据时动态高亮节点与连线
- 入口消息、出口消息、活跃会话等业务优先视图
- 技术事件明细和健康检查能力
- 可视化标签规则编辑器
- 内置同源 WebSocket 代理，减少浏览器直连网关时的握手问题
- 开源仓库整理完成，可直接 clone、安装、构建和运行

## Included

- 观察器模式与网关直连模式
- 会话摘要卡片
- 中文异常提示
- 图面布局与表单可读性优化
- MIT 许可证

## Notes

- 当前版本已经满足 V1 核心闭环，可单独运行、单独部署、单独验收
- 后续版本将继续围绕图面可读性、布局稳定性和业务字段丰富度迭代
