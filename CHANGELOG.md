# Changelog

## 2026-03-16

首个可公开使用的 V1 版本。

### Added

- 中文优先的 OpenClaw 网关链路可视化界面
- 观察器模式与网关直连模式
- 实时链路图与默认骨架图
- 最新入口消息、最新出口消息、活跃会话等业务视图
- 技术事件明细与健康检查能力
- 可视化标签规则编辑器
- 内置同源 WebSocket 代理
- 连接异常中文提示

### Improved

- 主视图优先展示业务信息，技术区域默认折叠
- 图面支持节点与链路高亮
- 表单控件、规则编辑区、图面布局持续优化
